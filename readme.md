# Blocket Web Crawler

A web crawler that follows relevant links and gathers data about items for sale on the swedish website blocket.

maedayBot0.1 retrieves:
- Ad title
- Price
- Description
- Ad URL [WIP]
- URL of the pictures [WIP]
